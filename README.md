## MDS

### Build & Run
The build and run process is based on `docker`.  
Make sure you have `docker-desktop` installed.  

> Windows  
```
Double click run-windows.bat
```

> Linux or MacOS  
```
$ sudo ./run-linux.sh
```

url: [`http://localhost`](http://localhost)
